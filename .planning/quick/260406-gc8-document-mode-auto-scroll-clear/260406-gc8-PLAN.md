   ```

   ```

3. 既存の conversionStore から setInput, setResult, setEditedResult も取得する（既にインポート済みの useConversionStore から）。TitleBar 内で destructure に追加:
   - conversionStore の destructure は現在 TitleBar の冒頭にある。ここに `setInput`, `setResult`, `setEditedResult` を追加する。

4. 「新しい会話」ボタンの onClick（現在 line 126-128）を以下に変更:
   ```typescript
   onClick={async () => {
     try {
       await newConversation();
       // フロントエンドのステートもクリア
       clearDocument();
       setInput('');
       setResult(null);
       setEditedResult(null);
     } catch (err) {
       console.error('Failed to start new conversation:', err);
     }
   }}
   ```
