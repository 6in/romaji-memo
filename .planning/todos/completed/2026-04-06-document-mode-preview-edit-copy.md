## Problem

長文書モードで変換した段落リストに対して以下ができない:

1. **変換済み段落を直接編集できない** — 出力テキストが `<p>` 読み取り専用で表示されている
2. **いきなりファイルダウンロードになる** — エクスポート前にまとめて確認できない
3. **全文をまとめてクリップボードにコピーできない** — ダウンロードしかない
4. **蓄積したテキスト全体に口調変換をかけられない** — ギャル語・ビジネス等への一括変換ができない

## Solution

### 1. 段落のインライン編集

`DocumentMode.tsx` の各段落の `<p>{p.output}</p>` を `<textarea>` に変更し、
`documentStore` に `updateParagraph(id, output)` アクションを追加する。

### 2. プレビューモード

「プレビュー」ボタンを追加し、押すと全段落を連結した1つの `<textarea>` に表示するビューに切り替わる。
プレビュー内のテキストも直接編集可能にする。

```
[ 段落リストビュー ] ←→ [ プレビュービュー (1つのtextarea) ]
```

`documentStore` に `previewText: string | null` を追加（null = 段落ビュー）。
プレビュー表示時は `paragraphs` を結合して初期値とし、編集はそのまま `previewText` に反映。

### 3. プレビューからクリップボードコピー

プレビュービューに「コピー」ボタンを追加。
`writeText(previewText)` でクリップボードにコピーする。

### 4. プレビューに口調変換

プレビュービューに `StyleSelector` + 「変換」ボタンを追加。
`previewText` 全体を選択中のスタイルで AI に送り、結果で `previewText` を上書きする。
（長文なので AI トークン上限に注意 — 段落単位で分割変換も検討）

## UX フロー

```
段落ビュー（各段落編集可）
  → [プレビュー] ボタン
  → プレビュービュー（全文textarea）
    → [コピー] / [口調変換] / [エクスポート]
    → [← 戻る] で段落ビューに戻る
```

## 実装優先順

1. インライン段落編集（`documentStore` に `updateParagraph`）
2. プレビュービュー（段落結合テキストの textarea 表示）
3. コピーボタン
4. 口調変換（最も複雑 — 長文分割が必要な場合あり）
