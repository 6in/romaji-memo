1. Run `npm create tauri-app@latest romaji-editor -- --template react-ts` in the parent directory of the workspace. If the workspace already IS `romaji-editor`, scaffold in a temp directory and move files. The scaffold creates: `package.json`, `tsconfig.json`, `vite.config.ts`, `src/main.tsx`, `src/App.tsx`, `src-tauri/` with `Cargo.toml`, `tauri.conf.json`, `src/main.rs`, `src/lib.rs`.

2. Install frontend dependencies:
   ```bash
   npm install tailwindcss @tailwindcss/vite
   npm install zustand @tanstack/react-query
   npm install lucide-react
   npm install sonner
   npx shadcn@latest init --defaults
   ```
   shadcn/ui init auto-detects Tailwind v4 and creates `components.json` plus `src/components/ui/` directory. Accept defaults.

3. Update `vite.config.ts` to include Tailwind v4 plugin:
   ```typescript
   import { defineConfig } from 'vite'
   import react from '@vitejs/plugin-react'
   import tailwindcss from '@tailwindcss/vite'

   export default defineConfig({
     plugins: [react(), tailwindcss()],
     clearScreen: false,
     server: { port: 5173, strictPort: true },
   })
   ```

5. Update `src-tauri/Cargo.toml` dependencies to match CLAUDE.md versions exactly. Do NOT use design doc section 6 versions (outdated):
   ```toml

   tauri = { version = "2", features = [] }
   tauri-plugin-clipboard-manager = "2"
   tokio = { version = "1", features = ["full"] }
   reqwest = { version = "0.13", features = ["json", "rustls-tls"] }
   serde = { version = "1", features = ["derive"] }

   rusqlite = { version = "0.39", features = ["bundled"] }

   thiserror = "2"
   ```
   Note: Do NOT include `tauri-plugin-global-shortcut` (Phase 2). Do NOT add `stream` feature to reqwest (D-05, no streaming in Phase 1).

6. Update `src-tauri/tauri.conf.json` window section (per design doc section 5.3 and D-16):
   ```json
   {
     "app": {
       "windows": [{
         "title": "Romaji Memo",
         "width": 420,
         "height": 600,
         "minWidth": 320,
         "minHeight": 400,
         "resizable": true,
         "alwaysOnTop": true,
         "decorations": false,
         "transparent": true
       }]
     }
   }
   ```
   Also ensure the `plugins` section includes `clipboard-manager`: `"plugins": { "clipboard-manager": {} }`.
   Ensure `security` > `csp` allows `tauri:` protocol and data URIs for icons.

7. Update `src-tauri/src/lib.rs` to register the clipboard plugin:
   ```rust
   pub fn run() {
       tauri::Builder::default()
           .plugin(tauri_plugin_clipboard_manager::init())
           .run(tauri::generate_context!())
           .expect("error while running tauri application");
   }
   ```
   Keep `main.rs` minimal (just calls `lib::run()`).

8. Update `src/App.tsx` to a minimal shell with QueryClientProvider:
   ```tsx
   import { QueryClient, QueryClientProvider } from '@tanstack/react-query'
   const queryClient = new QueryClient()
   function App() {
     return (
       <QueryClientProvider client={queryClient}>
         <div className="h-screen bg-background text-foreground">
           <p className="p-4">Romaji Memo</p>
         </div>
       </QueryClientProvider>
     )
   }
   export default App
   ```
