Create `README.md` at the project root with the following sections:

1. **Title and description:**
   - Romaji Memo — ローマ字入力をAIで日本語・英語・各種スタイルに変換するデスクトップアプリ
   - Floating utility that stays on top, converts romaji to Japanese/English via AI

2. **Prerequisites:**
   - Node.js 18+
   - Rust (rustup install)
   - macOS 12+ or Windows 10+

3. **Setup (per D-07 and D-10):**

   **macOS — Register API key in Keychain:**
   ```bash
   # Anthropic (Claude)
   security add-generic-password -s "romaji-memo" -a "anthropic" -w "sk-ant-YOUR_KEY_HERE"

   # OpenAI (optional)
   security add-generic-password -s "romaji-memo" -a "openai" -w "sk-YOUR_KEY_HERE"
   ```

   **Windows — Register API key in Credential Manager:**
   ```powershell
   # Anthropic (Claude)
   cmdkey /add:romaji-memo /user:anthropic /pass:sk-ant-YOUR_KEY_HERE

   # OpenAI (optional)
   cmdkey /add:romaji-memo /user:openai /pass:sk-YOUR_KEY_HERE
   ```

   **Ollama (Local — no API key needed):**
   - Install Ollama from https://ollama.ai
   - Pull a model: `ollama pull gemma3:12b`
   - Ollama must be running on `localhost:11434`

4. **Development:**
   ```bash
   npm install
   npx tauri dev
   ```

5. **Build:**
   ```bash
   npx tauri build
   ```

6. **Usage:**
   - Type romaji in the input area (spaces optional — AI infers word boundaries)
   - Select a conversion style (8 presets available)
   - Click "変換" or press Cmd/Ctrl+Enter
   - Click the copy icon to copy the result
   - Toggle always-on-top with the pin icon in the title bar
   - Toggle dark/light theme with the sun/moon icon
   - Open history with the bottom drawer button

7. **Provider Configuration:**
   - Default provider: Anthropic (Claude Haiku)
   - Edit `providers.json` in the app data directory to add/modify providers
   - API keys are stored in OS Keychain only — never in config files

8. **Tech Stack:**
   - Tauri 2 + React 19 + TypeScript
   - Tailwind CSS v4, shadcn/ui
   - Rust: rusqlite, keyring, reqwest
