   - `pub fn open_db(app_data_dir: &std::path::Path) -> Result<rusqlite::Connection, rusqlite::Error>` — opens `romaji-memo.db` in the given directory, sets PRAGMAs (`journal_mode=WAL`, `synchronous=NORMAL`, `foreign_keys=ON`), calls `run_migrations(&conn)`, returns connection.
   - `fn run_migrations(conn: &Connection) -> Result<(), rusqlite::Error>` — executes migration SQL from `migrations.rs`. Uses a `schema_version` PRAGMA or a `_migrations` check to avoid re-running.
   - Re-export `conversions` and `settings` submodules.

2. Create `src-tauri/src/db/migrations.rs`:
   - Contains `pub const MIGRATION_001: &str` with the full initial migration SQL wrapped in `BEGIN; ... COMMIT;`:
     - `CREATE TABLE IF NOT EXISTS conversions` (all columns from design doc section 4)
     - `CREATE TABLE IF NOT EXISTS custom_styles` (included for Phase 2 readiness)
     - `CREATE TABLE IF NOT EXISTS settings`
     - `CREATE INDEX IF NOT EXISTS idx_conversions_created ON conversions(created_at DESC)`
     - `CREATE INDEX IF NOT EXISTS idx_conversions_style ON conversions(style_id)`
     - `CREATE INDEX IF NOT EXISTS idx_conversions_pinned ON conversions(pinned DESC, created_at DESC)`
     - `CREATE VIRTUAL TABLE IF NOT EXISTS conversions_fts USING fts5(input, output, content='conversions', content_rowid='id')`
     - ALL THREE FTS5 triggers (CRITICAL — missing any trigger causes silent index corruption):
       - `conversions_ai` AFTER INSERT
       - `conversions_ad` AFTER DELETE
       - `conversions_au` AFTER UPDATE (delete old + insert new)
   - Uses `PRAGMA user_version` for migration versioning: check `user_version`, if 0 run migration, set to 1.

3. Create `src-tauri/src/state.rs`:
   ```rust
   use std::collections::HashMap;
   use std::sync::Arc;
   use tokio::sync::{Mutex, RwLock};
   use rusqlite::Connection;
   use crate::providers::ProviderAdapter;

   pub struct AppState {
       pub db: Arc<Mutex<Connection>>,
       pub providers: RwLock<HashMap<String, Arc<dyn ProviderAdapter + Send + Sync>>>,
       pub keychain_lock: Mutex<()>,
   }
   ```
   Note: `db` is `Arc<Mutex<Connection>>` so it can be cloned into `spawn_blocking` closures.

4. Update `src-tauri/src/lib.rs`:
